# Plataforma de Infraestructura Autoalojada

Infraestructura personal basada en KVM, Arch Linux y Docker.

Este repositorio es donde documento lo que hago con mi sistema: pruebas, errores, soluciones y configuraciones reales.

## Tecnologías
- KVM
- Arch Linux
- Docker

## Objetivo
Aprender y practicar administración de sistemas y cloud desde cero, construyendo mi propia infraestructura.
